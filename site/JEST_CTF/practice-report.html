1. Общая информация о проекте

Название проекта

"JESTь: Создание мини-набора CTF-задач по безопасности веб-приложений"

Цели и задачи проекта

Цель: разработать обучающий веб-сайт с набором CTF-задач по выявлению уязвимостей в веб-приложениях.

Задачи:

  • Разработка 6 учебных CTF-задач по основным типам веб-уязвимостей (XSS, SQLi, CSRF, IDOR, слабая аутентификация, поиск скрытых файлов).
  • Создание интерфейса сайта с навигацией, описанием задач, информацией о команде и контактами.
  • Интеграция с Telegram и GitHub.
  • Подготовка технического отчёта по результатам практики.

5. Описание CTF-задач

В рамках проекта были разработаны и интегрированы следующие задачи:

XSS Playground (Межсайтовый скриптинг)

Цель: научиться обнаруживать и эксплуатировать уязвимости XSS.

Что реализовано: форма ввода без экранирования вывода.

Учебные аспекты: типы XSS, защита через CSP и экранирование.

SQL Injector (SQL-инъекции)

Цель: изучить методы взаимодействия с SQL-базами через уязвимости ввода.

Что реализовано: форма авторизации без защиты от SQLi.

Учебные аспекты: UNION, error-based, blind SQLi; защита через параметризованные запросы.

AuthBuster (Слабая аутентификация)

Цель: исследовать слабые реализации систем аутентификации.

Что реализовано: система входа без ограничений на попытки.

Учебные аспекты: перебор паролей, защита через хэширование и MFA.

FileHunter (Поиск скрытых файлов)

Цель: освоить методы поиска конфиденциальных файлов на сервере.

Что реализовано: возможность находить robots.txt, .env и др.

Учебные аспекты: использование DirBuster, настройка прав доступа.

CSRF Trap (Подделка межсайтовых запросов)

Цель: понять механизм CSRF-атак и способы их предотвращения.

Что реализовано: форма изменения профиля без проверки источника.

Учебные аспекты: защита через токены CSRF и SameSite.

IDOR Leak (Прямой несанкционированный доступ к объектам)

Цель: обнаружить уязвимости типа IDOR.

Что реализовано: страница просмотра данных с изменяемым id в URL.

Учебные аспекты: проверка прав доступа, indirect object references.
