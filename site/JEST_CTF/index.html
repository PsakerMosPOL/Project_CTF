Кто мы?

Мы - CyberSmiths, команда из двух студентов (группы 241-371 и 241-352), создающая захватывающие CTF-задачи по безопасности веб-приложений. Наша цель - бросить вызов хакерам и помочь им отточить навыки в поиске уязвимостей. Готовы взломать наши задачи?

Наши CTF-задачи

  • XSS Playground - Экспериментируйте с межсайтовым скриптингом
  • SQL Injector - Погрузитесь в уязвимости SQL-инъекций
  • AuthBuster - Ломайте слабые механизмы аутентификации
  • FileHunter - Отыщите скрытые файлы на сервере
  • CSRF Trap - Поймайте уязвимость CSRF
  • IDOR Leak - Найдите данные других пользователей
